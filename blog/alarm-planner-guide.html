Build a Full Daily Planner with SeZZamE Alarms

July 3, 2026 7 min read SeZZamE Team

Most alarm apps treat alarms as individual, disconnected events. You set them one by one, manage them one by one, and if a schedule changes — a conference shifts by a day, a project timeline moves, a trip gets delayed — you adjust them one by one.

SeZZamE's Plans feature changes this completely. A Plan is a structured multi-day alarm schedule that moves as a single unit. Build it once, slide it when timing shifts, and every alarm adjusts automatically.

Here's how to use SeZZamE Plans to build a real daily planner.

What a Plan actually is

A Plan in SeZZamE is a named sequence of days, each populated with whatever alarms you need on that day. Think of it like a spreadsheet where each row is a day and each cell in that row is an alarm: its time, label, sound, snooze setting, timezone, and any attached location or link.

When you schedule a Plan to start on a specific date, SeZZamE activates those alarms on the right days automatically. When you need to move the Plan — say it was starting Monday but now starts Wednesday — you drag the start date and every alarm in every day shifts by exactly two days. No manual editing.

Practical use cases for Plans

Conference or event schedules

You're attending a three-day conference. Day 1: registration at 9 AM, keynote at 10 AM, networking lunch at 12:30 PM, afternoon sessions, dinner at 7 PM. Day 2: morning workshops, afternoon breakouts, speaker dinner. Day 3: closing keynote, checkout.

Set all of these up in a Plan with the conference start date. If the conference gets rescheduled by a week, drag the Plan's start date — done. All 15+ alarms shift together.

Travel itineraries

Build a Plan for your trip: Day 1 flight alarm at 4:30 AM, hotel check-in reminder at 3 PM. Day 2 tour pickup at 8 AM, museum opening at 10 AM. Day 3 checkout alarm, return flight. Each alarm can be pinned to the local timezone of your destination so they fire correctly regardless of where your phone's clock is set.

Work sprints or project phases

Set up a weekly sprint schedule as a Plan: daily standup at 9 AM, mid-day focus block at 1 PM, standup wrap at 5 PM, end-of-sprint review on Friday at 3 PM. When a sprint pushes by a few days, move the whole Plan. Recurring Plans let you repeat this weekly without resetting anything.

Family routines

Build a school week Plan: kids wake-up at 6:30 AM, school bus at 7:45 AM, after-school pickup at 3:15 PM, homework time at 4 PM, dinner at 6:30 PM, bedtime at 8:30 PM. Share the Plan via SeZZamE's sharing feature with your partner so you're both running the same schedule.

How to create a Plan in SeZZamE

  1. Open the Plans tab in SeZZamE (the calendar icon in the tab bar).
  2. Tap the + button to create a new Plan.
  3. Give the Plan a name — "Berlin Conference," "Summer Sprint," "School Week."
  4. Set a start date. This is the date Day 1 of the Plan will be active.
  5. Add days using the + Day button. Each day gets its own alarm set.
  6. For each day, tap + Alarm to add alarms — set the time, label, sound, and any options.
  7. Optionally add a repeat to make the Plan recur — weekly, monthly, quarterly, or yearly.
  8. Save the Plan. SeZZamE activates the alarms on the right calendar days.

Shifting a Plan when timing changes

In the Plans tab, tap and hold a Plan card to enter edit mode, or open the Plan and adjust the start date. Drag the date forward or backward. SeZZamE recalculates all alarm dates immediately — you can see the preview before saving. Tap Save and you're done.

This single gesture replaces what would otherwise require opening and editing every alarm individually.

Custom repeat patterns

SeZZamE supports repeat patterns beyond the standard daily/weekly that the built-in Clock app offers:

  • Weekly — runs the same days every week
  • Monthly — repeats on the same date each month, or on patterns like "first Monday of the month"
  • Quarterly — useful for business reviews, tax reminders, or quarterly check-ins
  • Yearly — annual reminders that you want to appear at exactly the right time each year

For truly custom patterns, SeZZamE also supports cron-style schedule definitions — so you can express things like "every 10 days" or "the last Friday of every month."

Combining Plans with other SeZZamE features

Each alarm inside a Plan has access to every SeZZamE alarm feature:

  • Timezone pinning — each alarm can fire in its own timezone. Build a travel Plan where Day 1 alarms fire in EST and Day 3 alarms fire in GMT+1 as you cross into a new zone.
  • Custom sounds — give each alarm in a Plan a distinct sound so you know which one fired without looking.
  • Location & link attachments — attach the address and the booking link for each activity in a travel Plan so everything is one tap away when the alarm fires.
  • Motivation screens — set a different motivational message for each day's first alarm to set the tone for that day.

Sharing a Plan

Any SeZZamE Plan can be exported as a link or QR code. The person you share it with opens the link in their SeZZamE and gets an identical Plan — all the days, all the alarms, all the options. One share, full synchronisation. This makes SeZZamE the ideal alarm app for couples, families, event teams, or anyone running a shared schedule.
